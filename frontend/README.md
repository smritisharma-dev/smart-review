# Smart Review

A MERN stack project to manage Google and core business reviews.

## Features
- Fetch business reviews  
- Approve, reject, pending status  
- Dashboard with pie charts  
- Admin panel for managing reviews  

## Tech Stack
- React (Vite)  
- Node.js  
- Express  
- MongoDB  

## How to Run

### Frontend
```bash
cd frontend
bun install
bun run dev 
```
### Backend
```bash
cd backend
npm install
node server.js
```
